================================================================================
  🔄 HOW IT WORKS
================================================================================

1. DEVELOPER PUSHES CODE TO GITHUB
   ↓
2. GITHUB WEBHOOK TRIGGERS JENKINS
   ├─ Code checkout
   ├─ Run tests (pytest + coverage)
   ├─ Security scanning (SonarQube, OWASP, Bandit)
   ├─ Build Docker image
   ├─ Scan image for vulnerabilities (Trivy)
   ├─ Push to DockerHub: username/paperintel:BUILD_ID
   └─ Update ArgoCD manifests
   ↓
3. ARGOCD DETECTS CHANGES
   ├─ Watches Git repository
   ├─ Detects new image tag
   ├─ Auto-syncs manifests to K3s
   └─ Updates deployment
   ↓
4. KUBERNETES AUTO-DEPLOYS
   ├─ Pulls new image from DockerHub
   ├─ Rolls out new pods (rolling update)
   ├─ Maintains 3 replicas minimum
   ├─ Auto-scales up to 10 pods if needed
   └─ Health checks every 5-10 seconds
   ↓
5. MONITORING WATCHES EVERYTHING
   ├─ Prometheus collects metrics
   ├─ Grafana visualizes dashboards
   ├─ AlertManager sends Slack alerts
   └─ Node/Container exporters track resources
   ↓
6. APPLICATION RUNNING IN PRODUCTION
   ├─ Fully automated
   ├─ Continuously monitored
   ├─ Auto-scaling & self-healing
   └─ Easy rollback if needed
